# Welcome to BudgetBuddy

BudgetBuddy is a personal and social finance web application designed to help users track their income, expenses, and savings in customizable pay periods. It encourages healthy financial habits through goal setting, smart nudges, visualizations, and friendly social features, while keeping a light gamified experience to motivate consistent usage.

---

## 🌟 Features

- **Custom PayCycles**: Define periods that align with your pay schedule (weekly, biweekly, monthly, etc.).
- **Income & Expense Tracking**: Record all earned income, savings, and expenses.
- **Custom Categories & Goals**: Users can define categories and set goals as a percentage of income or flat amounts.
- **Automatic Rollover**: Unused category balances can roll over to the next PayCycle.
- **Recurring Transactions**: Schedule recurring incomes or expenses.
- **PayCycle Reality Check**: Compare actual spending vs planned budget.
- **Visual Timelines**: See PayCycle history with planned vs actual spending visually.
- **Long-Term Goals**: Track progress toward bigger financial goals.
- **End-of-PayCycle Ritual**: Summarized review of achievements and goal completion.
- **Lightweight Gamification**: Streaks, badges, and progress indicators for motivation.
- **Social Features**: Share progress with friends, compare goal completion percentages (privacy-conscious).

---

## 🛠️ Tech Stack

- **Frontend**: React, TypeScript, Vite, Tailwind CSS, React Router, TanStack Query, React Hook Form, Zod, Chart.js / Recharts  
- **Backend**: Python 3.12, FastAPI, SQLAlchemy 2.0, Pydantic, Alembic, SQLite  
- **Hosting / Deployment**: Single VPS 
- **Optional AI Features**: For smart nudges, automated categorization, and goal recommendations
